# EventHub

EventHub is a React-based event management application that allows users to browse events, view event details, book tickets, and manage their bookings.

## Features

- Browse all available events
- View detailed event information
- Filter events by category
- Book tickets for events
- Calculate total booking price
- Store bookings using Local Storage
- View all bookings
- Delete bookings
- Responsive UI using Material UI

## Technologies Used

- React.js
- React Router DOM
- Material UI (MUI)
- JavaScript (ES6+)
- Local Storage

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/EventHub-Frontend.git
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
